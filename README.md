# casual-hr-system
SEF Assignment
This assignment is a program for a casual HR system.

## Installing
just zip the above folder and import it into Netbeans, or just drop it in your NetbeansProjects folder.

## Running
To run this program in it's current release version, navigate to the 'CasualHRSystem/dist' folder in the command line and execute the following command.

	java -jar CasualHRSystem.jar

You can login using the following test user credentials (automatically added to the database on initial execution or deletion of the database):
